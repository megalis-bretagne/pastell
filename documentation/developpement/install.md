# Installation de l'environnement de développement avec Docker

cd [SOURCE DE PASTELL]

Il faut configurer l'environnement :
```
vi .env
```
un exemple est fourni dans .env_exemple

```bash
docker-compose up -d
```
 
Accès au site : 
- https://localhost:8443
- login : cf .env
- mot de passe : cf .env

Accès à PhpMyAdmin:
- http://localhost:8001
