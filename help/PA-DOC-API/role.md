# Rôles

# Lister les rôles

Récupérer la liste des groupes.

```
GET /role
```

Exemple de réponse:

```json
[
    {
        "role": "admin",
        "libelle": "Administrateur"
    },
    {
        "role": "fournisseur",
        "libelle": "fournisseur"
    },
    {
        "role": "gestion_actes",
        "libelle": "Gestionnaire ACTES"
    },
    {
        "role": "r\u00f4le",
        "libelle": "mon r\u00f4le"
    },
    {
        "role": "gestion_pes",
        "libelle": "Gestionnaire PES"
    }
]
```
